3. **Product Performance**: Sales volume by product, category, supplier
4. **Employee Performance**: Sales by employee with territory information
5. **Regional Analysis**: Sales performance by region and territory

JOIN bottlenecks identified:
- Orders → Order Details → Products → Categories/Suppliers (4+ table join)
- Orders → Customers → Customer Demographics
- Orders → Employees → Employee Territories → Territories → Regions

## 3. Evaluation of Denormalization Techniques

Based on the schema and workload, these techniques are most applicable:

1. **Adding Redundant Attributes**: High value - eliminates multiple JOINs in sales analysis
2. **Adding Derived Attributes**: High value - precomputed aggregations for analytical queries
3. **Collapsing Relations**: High value - especially for frequently joined hierarchies
4. **Vertical Partitioning**: Medium value - only beneficial if certain columns are rarely accessed
